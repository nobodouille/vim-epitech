# vim-epitech

Vim plugin for Epitech students rewritten slighty in lua.


## Installation

### Vim-plug

```vim
Plug 'Sigmanificient/vim-epitech'
```

### Packer

```vim
use "Sigmanificient/vim-epitech"
```

### lazy.nvim

```vim
{ "Sigmanificient/vim-epitech" }
```
